endpoints


below are described the rest endpoints available that you can use to make requests.



{base-url}/

Desc : api welcomes you.


Response Status: 200

{base-url}/todos

Desc : this endpoint will retrieve all the todos. the todos are stored inside object and them inside an array.


Response Status: 200

{base-url}/todos/new

Desc : this endpoint will create a new todo item and push it to array that stores all your todos.


Request Body/Raw

Response Status: 201

{base-url}/todos/:id

Desc : this endpoint will help you update an already existing todo item taking in todo id as query parameter inside the url.


Request Body/Raw

Response Status: 200

{base-url}/todos/:id

Desc : this endpoint will allow you to delete existing todo item with it's id taking in as query param.


Response Status: 200
